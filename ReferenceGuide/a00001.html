Detailed Description

This class contains information about a specified license in the license chain.

Information includes license type, product type (Symantec Endpoint Protection or Symantec Network Access Control), number of purchased seats, and expiration date.

Note that the number of used seats is reported only as a total of all used seats. You must therefore call LicenseSummaryInfo to get the number of used seats.

int com.symantec.sepm.webservice.admin.license.LicenseInfo.getLicenseType ( )

This license type is the type of the individual license for which detailed information is provided.

See also:
LicenseSummaryInfo
Returns:
an int value that represents the individual license type.

Possible values and their meaning:

0 = LICENSE_INFO_TRIAL. The license type is a trial license.

1 = LICENSE_INFO_UPGRADE. The license type is an upgrade license.

2 = LICENSE_INFO_PURCHASED. The license type is a purchased license.
