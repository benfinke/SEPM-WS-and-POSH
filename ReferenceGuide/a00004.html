Detailed Description

Provides summary information about the state of licensing for a specified Symantec Endpoint Protection Manager.

Information includes the number of used seats and purchased seats, the next renewal deadline, whether licensing is in an over deployed state, and whether any license is about to expire (will expire within thirty days). An array of child LicenseInfo objects contains information about each license in the license chain.

The license type for LicenseSummaryInfo represents the type of the most current license. For example, if a trial license exists and is not expired, and a license is subsequently purchased, two LicenseInfo objects exist, one for the trial license and one for the purchased license. The LicenseSummaryInfo object in this case contains only a license type for the purchased license (SEPE_PAID or SEPSB_PAID).

int com.symantec.sepm.webservice.admin.license.LicenseSummaryInfo.getLicenseType ( )

This license type is the type of the most current valid license.

Any older licenses or invalid licenses are not accounted for here.

See also:
LicenseInfo
Returns:
an integer value that represents the currently valid license type.

Possible values and their meaning:

0 = LICENSE_STATE_SEPSB_TRIAL. A trial license for Symantec Endpoint Protection Small Business Edition.

1 = LICENSE_STATE_SEPSB_PAID. A paid license for Symantec Endpoint Protection Small Business Edition.

2 = LICENSE_STATE_SEPE_TRIAL. A trial license for Symantec Endpoint Protection Enterprise Edition.

3 = LICENSE_STATE_SEPE_UPGRADE. An upgrade license for Symantec Endpoint Protection Enterprise Edition.

4 = LICENSE_STATE_SEPE_PAID. A paid license for Symantec Endpoint Protection Enterprise Edition.

5 = LICENSE_STATE_UNKNOWN. The license type is unknown (licensing error). For suggestions about how to fix, see the Symantec Endpoint Protection documentation or other resources described in the Integration Guide.

String com.symantec.sepm.webservice.admin.license.LicenseSummaryInfo.getNextRenewalDeadline ( )

Returns the renewal deadline of the next license that is about to expire.

Returns:
the next renewal deadline in UTC format.

Returned using the following logic:

  1. If a license that is about to expire will cause over-deployment of both Symantec Endpoint Protection and Symantec Network Access Control, the nearest renewal deadline is returned.
  1. Else if a license that is about to expire will cause over-deployment of Symantec Endpoint Protection only, the renewal deadline for that license is returned.
  1. Else if a license that is about to expire will cause over-deployment of Symantec Network Access Control only, the renewal deadline for that license is returned.
  1. Else Symantec Endpoint Protection Manager is completely out of compliance, because all existing licenses are expired. So the method returns null.
