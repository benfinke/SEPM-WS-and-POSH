Detailed Description

Provides helper functionality for the ClientService API.

This class validates input parameters and populates transfer objects and filter objects. Additional validation is provided by other classes.

See also:
IpValidator
HostNameValidator
GroupNameValidator

static String com.symantec.sepm.webservice.client.ClientServiceHelper.validateAndReconstructGroupPath ( String  groupPath) throws InvalidArgumentServiceException [static]

Validates the group path parameter of the ClientService.getGroupsByName method.

The group path is reconstructed according to the following rules:

  • all forward slashes ("/") are replaced with backslashes ("\")
  • any forward slashes or backslashes at the beginning or end of the group path are removed
  • any duplicate backslashes in the group path are removed
See also:
GroupNameValidator
Parameters:
groupPathgroup path to validate
Returns:
reconstructed group path
