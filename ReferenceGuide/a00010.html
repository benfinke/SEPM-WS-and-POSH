Detailed Description

Provides helper functionality for the CommandService API.

This class validates input parameters and populates transfer objects and filter objects. Additional validation is provided by other classes.

See also:
GroupNameValidator
GUIDValidator See also the other classes in the validation package.

Creates a transfer object for the specified command.

The timeLimit parameter is ignored if the specified command does not take timeLimit as a parameter. In this case, pass any value.

For information about Symantec Endpoint Protection Manager sites, see the product documentation.
