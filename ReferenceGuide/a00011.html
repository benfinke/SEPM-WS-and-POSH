PagingOptions com.symantec.sepm.webservice.client.command.CommandServiceImpl.createPagingOptions ( )

Creates a PagingOptions object.

This method is provided as a convenience for developers using PowerShell or other scripting languages. It lets the caller create a default object that can be initialized and passed as an input parameter to other web services. Developers who use higher-level languages should be able to create this object directly, or they can use this helper method.
