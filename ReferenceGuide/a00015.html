Detailed Description

Provides a list of child Computer objects that match the search parameters of the ClientService.getComputersbyIP and ClientService.getComputersbyHostName methods.

These objects provide the computer GUID that is passed to the methods of other web services.

Also provides methods to help manage large result sets. For example, if the getMoreResultAvailable method returns true, then the next set of results can be retrieved by using the offset (index) provided by Context().
