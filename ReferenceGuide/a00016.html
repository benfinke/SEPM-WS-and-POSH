boolean com.symantec.sepm.webservice.client.Group.isPolicyInheritanceEnabled ( )

Returns true if this group is configured to inherit its policy from one of its ancestor groups.

If inheritance is enabled, the group's policies cannot be changed. Returns false if inheritance is disabled for this group. If inheritance is disabled, the group's policies can be changed.

Returns:
the policyInheritanceEnabled state
void com.symantec.sepm.webservice.client.Group.setPolicyInheritanceEnabled ( boolean  policyInheritanceEnabled)

Sets the policy inheritance state for this object to match the persistent state of this group.

Web service client applications should not call this method. This method modifies only the local object. Any changes are not sent back to the management server.
