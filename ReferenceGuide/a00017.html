Detailed Description

Provides a list of child Group objects that match the search parameters of the ClientService.getGroupsByName method.

These objects provide the group ID that is passed to the methods of other web services.

Also provides methods to help manage large result sets. For example, if the getMoreResultAvailable method returns true, then the next set of results can be retrieved by using the offset (index) provided by Context().
