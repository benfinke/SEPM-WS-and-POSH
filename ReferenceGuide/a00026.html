Detailed Description

The PagingOptions class creates an object that provides pagination and sorting information to be used together with a Context object to simplify retrieving large result sets.

For example, the lists of computer GUIDs or group names that are returned by the methods of the ClientService can be very large. A PagingOptions object can be used to limit the number of records returned by a single call to these methods.

Example:

 // Using the default constructor
PagingOptions defaultPagingOptions = new PagingOptions();
// Specifying the page size
int pageSize = 100; // pageSize is the number of records to be fetched in a single method call
 // valid value is between 1 and 1000 (inclusive)
 PagingOptions pagingOptions = new PagingOptions(pageSize);
