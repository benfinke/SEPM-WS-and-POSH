Detailed Description

Helper class that provides methods to check the access rights of the current logged in user.

Many calls to Symantec Endpoint Protection web services can be completed only if the logged-in user has access rights to perform the specified action in Symantec Endpoint Protection Manager.

For more information about access rights, see the Symantec Endpoint Protection documentation.

Checks the access rights for the current logged in user by checking the web services URL.

System administrators and domain administrators by definition have sufficient access rights to all web services functionality and therefore pass this check. This method provides a check for all the methods of a single web service.

Parameters:
serviceNamethe name of the web service. Mapped to a URL to leverage basic URL checking logic.
permissionManageran instance of the PermissionManager that performs the check
Exceptions:
InsufficientAccessServiceExceptionif the current logged in user does not have sufficient rights to access the web service endpoint
