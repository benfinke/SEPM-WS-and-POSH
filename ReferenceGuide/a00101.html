Detailed Description

Provides the web service interface and implementing classes for reporting on the Symantec Endpoint Protection license state.

The license web service returns information about summary license state and about the state of individual licenses. For more information about how Symantec Endpoint Protection licensing works, see the Integration Guide.

Note that the product can be managed with more than one purchased license. Each purchased license is associated with a specified number of seats.

Trial licenses and upgrade licenses are unlimited, but expire after a predefined period of time. A purchased license contains a specified number of seats and an expiration date.

If licenses are purchased before a trial license or upgrade license expires, the trial license or upgrade license is removed, and only the purchased license(s) are reported on.
