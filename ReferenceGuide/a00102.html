Detailed Description

Provides the web service interface and implementing classes for getting lists of computer GUIDS or Endpoint Protection group IDs.

Computer GUIDs can be retrieved by looking up IP addresses, host names, or domain names. Group IDs can be retrieved by using the group name or the asterisk wild card character (*), and child group names can be excluded or included.

Returning large result lists can be managed by using the PagingOptions object and the Context object.

The returned lists of ComputerResult objects can then be passed as parameters to the methods of the CommandService API. Lists of ComputerResult objects or GroupResult objects can be passed as parameters to the methods of the PolicyService API.

For more information about Endpoint Protection groups and lists of client computers, see the Endpoint Protection documentation.
