Detailed Description

Provides the web service interface and implementing classes for running commands on specified lists of Symantec Endpoint Protection clients and returning the results of the commands.

Commands include updating content, scanning for threats, and more.

See also:
ClientService

For more information about running commands on clients, see the Integration Guide and the Symantec Endpoint Protection documentation.
