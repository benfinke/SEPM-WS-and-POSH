Detailed Description

Provides the web service interface and implementing classes for checking the status (date and revision) of content updates on the Symantec Endpoint Protection management server and on Symantec LiveUpdate servers.

Also provides functionality for running LiveUpdate on the management server.

For more information about running LiveUpdate and about LiveUpdate content, see the Endpoint Protection documentation.
