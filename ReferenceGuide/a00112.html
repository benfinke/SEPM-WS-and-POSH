Detailed Description

Provides the web service interface and implementing classes for importing Symantec Endpoint Protection policies and optionally applying the policies to specified groups or locations.

To import a policy using web services, the policy must first be created and exported in the management console. The policy can be assigned when it is imported, or it can be saved to the policy library for assignment at a later time.

Policies can also be assigned separately. Policies to assign can be imported from another Endpoint Protection management site, or they can be the locally installed default policies, or they can be locally created custom policies.

The logged-in user making calls to these web services must have administrator access rights to manage the following Symantec Endpoint Protection management entities:

  • The domain to which groups that will receive the policy belong
  • The specified groups that will receive the policy
  • The protection technologies that are included in the policy

Symantec Endpoint Protection uses the concept of locations within groups. A policy can include location-specific settings, and the Policy web services let you assign policies to specified locations.

For more information about access rights, policies, groups, and locations, see the Endpoint Protection documentation.
