What's new in this release

The following new functionality is available in this release to support remote monitoring and management of Symantec Endpoint Protection clients and management servers.

  • License state reporting on the management server by means of web services.

    Previous releases provided license state reporting by means of the Windows Event Log. You can now also monitor license state with the new web service APIs.

    See License reporting.

  • Content state reporting on the management server by means of web services.

    Previous releases provided content state reporting by means of the Windows Event Log. You can now also monitor content state with the new web service APIs.

    See Checking content from LiveUpdate on the server.

  • The ability to run LiveUpdate on the management server.

    See Running LiveUpdate on the server.

  • Remediation of client state by means of web services. Client remediation can be performed remotely in either of two ways:

    • By issuing commands to lists of clients.

      With commands, you can perform immediate remediation of such states as Infected or Out of date.

      See Running commands on clients.

    • By managing the policies that are delivered to the client.

      You can assign policies to groups in the current Symantec Endpoint Protection Manager site or import and assign policies that are exported from another Symantec Endpoint Protection Manager site.

      See Assigning policies to clients.

See Remote management and monitoring of Symantec Endpoint Protection.
