Installing and configuring Symantec Endpoint Protection Manager web services for remote management

By default, Symantec Endpoint Protection Manager web services are installed and enabled when you install Symantec Endpoint Protection Manager. The default port for remote management web services communication is 8446. To change this port, run the Management Server Configuration Wizard, choose Custom configuration, and change the port number. If you run Symantec Endpoint Protection Small Business Edition, you cannot change this port.

If you develop a remote management web services client application, make sure that your application provides an easy way for your customers to update the port number if they change it.

Note:

Symantec Endpoint Protection Manager provides two sets of web services. To customize the appropriate port, make sure that you choose the field labeled Remote management web services port.

When you or your customers configure your server environment, make sure that the web services port can accept incoming TCP traffic.

A script is provided to disable or re-enable web services. You can also use the script to enable or disable the earlier set of web services that were written for integration with internal Symantec tools.
