Using OAuth with Symantec Endpoint Protection Manager web services

Symantec's implementation of the OAuth 2.0 protocol requires an administrator account on the Symantec Endpoint Protection Manager server. This account must be a Symantec Endpoint Protection Manager account. Authentication with an LDAP/Active Directory server account is not supported.

You may find it simpler to get started with OAuth and Symantec Endpoint Protection Manager web services by running the sample PowerShell scripts that are provided in the SDK. See Using the sample PowerShell scripts.
