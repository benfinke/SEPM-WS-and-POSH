Managing the Symantec Endpoint Protection client with web services

Remote management web services let you remediate client state issues that are detected with remote monitoring tools. See Monitoring client state and activity.

You can perform immediate remediation by running the appropriate command on the client. See Running commands on clients.

If a different security policy should be applied to a group of clients, then you can assign a new policy by using the PolicyService web service. See Assigning policies to clients.
