Web services are provided for managing the following functions of the management server remotely:

Table:

Function

Description

See License reporting.

See Checking content from LiveUpdate on the server.

Running LiveUpdate

See Running LiveUpdate on the server.

Note:

The customer must be logged in to the remote management application with a Symantec Endpoint Protection Manager account that has the appropriate access rights on the management server. Only a Symantec Endpoint Protection Manager administrator that has System Administrator access rights can manage licenses.

Content state events and license state events are also logged in the Windows Event Log, as in previous versions.
