Running LiveUpdate on the server

The new LiveUpdateService API provides a method for running LiveUpdate on Symantec Endpoint Protection Manager from a remote management web service client. This method lets you update content on the management server whenever the content is reported as out-of-date. See Checking content from LiveUpdate on the server.

Any of the following Symantec Endpoint Protection Manager administrator accounts is required to run LiveUpdate:

  • System administrator

  • Domain administrator

  • Limited administrator with full access rights to manage the site

    A limited administrator with fewer access rights cannot run LiveUpdate.

For more information about running LiveUpdate on the management server and about Symantec Endpoint Protection client content downloads, see the Symantec Endpoint Protection documentation.

For more information about running LiveUpdate with web services, see the Reference Guide in this SDK.

See Managing Symantec Endpoint Protection Manager with web services.
