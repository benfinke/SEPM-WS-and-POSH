License reporting

This version of Symantec Endpoint Protection provides a web service for monitoring the status of licenses on Symantec Endpoint Protection Manager. You can also monitor license status on the management server by using the Windows Event Log, as in previous versions, but we recommend that you use the new web service.

The LicenseService API provides functionality for reporting on current license state (whether licensing is up to date) and on the details of individual licenses in the license chain. Current license state is derived from the state of the most recent license in the chain.

See Managing Symantec Endpoint Protection Manager with web services.
