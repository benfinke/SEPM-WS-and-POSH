VisualBasic script to detect management server version

You may find the following sample script useful as a model for reporting on the version number of the management server.

Example version number: 12.1.1047.347

	'set  computer name - replace variable with appropriate value
	Computer = "."

	'obtain collection of Windows Installer packages
	Set MSIapps = GetObject("winmgmts:{impersonationLevel=impersonate}!\\" & Computer &_ 
	"\root\cimv2").ExecQuery("select * from Win32_Product where Name like 
'%Symantec Endpoint Protection Manager%'")

	'obtain number of program in collection
	AppList = AppList & MSIapps.Count & " Symantec packages installed:" & VBCRLF & "------" & VBCRLF

	'enumerate the names of the packages in the collection
	For each App in MSIapps
		AppList = AppList & App.Name & VBTab & App.Version & VBCRLF
	Next

	'display list of packages
	Wscript.Echo AppList
