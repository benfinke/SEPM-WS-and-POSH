Licensing events

The following events are written to the Windows Event Log on the Symantec Endpoint Protection management server.

Note:

EventIDs 7103 and 7113 (out of compliance with licenseing requirements) are written in the case of any expired license type: purchased, trial, or upgrade. These events are also written in the case where a current license is deleted on the Symantec Endpoint Protection management server. Any scripts written to monitor licensing status should therefore examine the timestamp of the event first, not the eventID.

See About Symantec Endpoint Protection licensing.

The licensing message text field contains the following subfields, in addition to the message text prefix that is specific to each eventID.

Table: Licensing message text details

Subfield label

Subfield description

Total licensed seats:

The total number of available seats in all licenses.

Total seats used:

The total number of used seats in all licenses.

License type:

ENUM. 0 = Purchased (per license), 1 = Trial, 2 = Upgrade

Note:

This subfield and the following two subfields repeat for each license.

Seats in this license:

The number of seats in the license specified by the license type subfield. For trial and upgrade licenses, this value is always Unlimited, even though the license may have expired. In the case of expired trial or upgrade licenses, the Total licensed seats field displays 0.

Expiration date for this license:

The expiration date for this license, in UTC complete date format.
