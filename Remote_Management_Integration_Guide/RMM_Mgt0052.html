Operational state reference

The following table documents the publicly supported registry subkeys that report operational state data on the Symantec Endpoint Protection client.

All registry subkeys are written to the following location: HKLM\SOFTWARE\Symantec\Symantec Endpoint Protection\CurrentVersion\Public-Opstate. New subkeys in this release are marked with an asterisk (*).

Warning:

All registry subkeys and values for Symantec Endpoint Protection should be treated as read-only.

See Registry key location changes in version 12.1.11xx.

Table:

Subkey

Type

Possible values

Used for

ASRunningStatus

REG_DWORD

1 == running

2 == disabled

Registers whether Virus and Spyware Protection is enabled or disabled.

Note:

This subkey appears to be redundant with the following subkey.

AVRunningStatus

REG_DWORD

1 == running

2 == disabled

Registers whether Virus and Spyware Protection is enabled or disabled.

ComputerID*

REG_SZ

ComputerIDString

The GUID of the client computer.

ContentDownloadHealth*

REG_DWORD

1 == success

2 == failure

The current status of the content download (virus definitions and other content) to the client.

DeployPreviousVersion

REG_SZ

4-part version string

Registers the four-part version number of the Symantec Endpoint Protection client software that was previously installed on the client computer.

DeployRunningVersion

REG_SZ

4-part version string

Registers the four-part version number of the Symantec Endpoint Protection client software that is currently installed on the client computer.

DeployStatus

REG_DWORD

See table below

Registers details about the status of client software download, installation, upgrade, or patch.

DeployTargetVersion

REG_SZ

4-part version string

Registers the four-part version number of the Symantec Endpoint Protection client software that is planned for future installation on the client computer.

FWRunningStatus

REG_DWORD

1 == running

2 == disabled

Registers whether firewall protection is enabled or disabled.

Infected

REG_DWORD

0 == clean

1 == infected

Registers whether the client computer is infected with one or more risks that are detected by Virus and Spyware Protection.

LastServerIP

REG_SZ

IP address

Registers the IP address of the most recent Symantec Endpoint Protection management server that the client connected to.

LastSuccessfulScanDateTime*

REG_SZ

Most recent successful scan time

The timestamp of the most recent scan that successfully ran on the client.

LatestVirusDefsDate*

REG_SZ

Latest content date

The timestamp of the most recent content that was downloaded to the client, in Pacific Standard or Daylight Time.

LatestVirusDefsRevision*

REG_DWORD

Latest content revision number

The revision number of the most recent content that was downloaded to the client.

RebootReason

REG_DWORD

0 == NO_REBOOT

1 == REMEDIATION

2 == PRODUCT_PATCH

3 == CONTENT_UPDATE

4 == INSTALL_COMPLETION

5 == SPC_COMMAND

6 == INSTALL_CATASTROPHIC_FAILURE

Registers the reason for a restart of the client computer.

snac_enabled

REG_DWORD

0 == disabled

1 == running

Registers whether Symantec Network Access Control is enabled or disabled.
